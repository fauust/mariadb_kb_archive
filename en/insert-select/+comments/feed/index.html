MariaDB Knowledge Base Comments for: INSERT SELECThttps://mariadb.com/kb/en/insert-select/+comments/feed/en-usSat, 17 Jun 2023 17:15:33 +0000INSERT SELECT with INIVISIBLE fieldshttps://mariadb.com/kb/en/insert-select/+comments/4862<p>If you're doing an INSERT SELECT and the destination table has any INVISIBLE fields and you try to pull data into those fields, the query will fail due to field count mismatch (at least as of 10.3.20). </p> <p>This can be an issue when migrating data. For example: INSERT INTO new_table SELECT old_table.*, old_table.invisible_field FROM old_table;</p> <p>Hopefully something like the following will be permitted in the future: INSERT INTO new_table SELECT * FROM old_table WITH INVISIBLE;</p> Ryan Leadenhamhttps://mariadb.com/kb/en/insert-select/+comments/4862
