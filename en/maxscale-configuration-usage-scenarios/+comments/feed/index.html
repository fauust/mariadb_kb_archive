The service user given in the configuration file is used for querying user information from the servers so that MaxScale can perform preliminary authentication. The monitors also have their own credentials which could explain why <code>show processlist</code> displays open connections created by the monitor user.</p> <p>Here's example output of me connecting to a local server with the <code>markusjm</code> user. The server is monitored by MaxScale running on localhost with the <code>maxuser</code> user.
